# 스네이크 게임 (Snake Game)

JavaScript, HTML5 Canvas로 만든 클래식 스네이크 게임입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 플레이할 수 있습니다.

## 기능

- 방향키(↑ ↓ ← →)로 뱀 조작
- 난이도 선택 (쉬움 / 보통 / 어려움 / 매우 어려움)
- 일시정지 / 재개
- 현재 점수 및 최고 점수 표시 (최고 점수는 `localStorage`에 저장)
- 게임 오버 후 다시 시작

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 게임 화면 구조 (스코어보드, 캔버스, 오버레이 UI) |
| `style.css` | 게임 스타일 |
| `main.js` | 게임 로직 (뱀 이동, 충돌 판정, 점수 계산 등) |

## 조작법

- **방향키**: 뱀 이동
- **게임 시작 / 다시 시작 버튼**: 게임 시작
- **일시정지 버튼**: 게임 일시정지 및 재개
